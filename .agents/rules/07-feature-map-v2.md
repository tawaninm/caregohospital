# 07 Feature Map v2

VoiceMed v0.2.0 must-have features:

- Landing page.
- Pricing.
- Mock checkout.
- Onboarding.
- Family dashboard.
- Elder profiles.
- Bot Settings.
- Care templates.
- Call history.
- Alerts.
- Reports.
- Billing.
- Settings.
- Patch Log.

All primary UI should be B2C family/caregiver oriented.
