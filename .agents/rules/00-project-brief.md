# 00 Project Brief

VoiceMed is a B2C AI Voice Companion subscription platform for families caring for elderly relatives.

Primary users:

- Family account owner / payer.
- Shared caregiver.
- Viewer family member.

Core flow:

Landing -> pricing -> mock checkout -> onboarding -> elder profile -> bot setup -> daily voice/chat follow-up -> summary -> alert/report.

B2B/hospital is future optional channel only.
